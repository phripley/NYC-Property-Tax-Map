NYC Property Tax Map

New York's property tax system is unfair. Similar properties regularly recieve hugely different tax bills. Wealthy people on average pay lower tax rates than the poor. Learn about New York's flawed property tax system and how we can fix it.

This map shows the property taxes levied on each property in New York City.

Tap on a property to see how much tax it pays per year.

Click or tap on a property on the map to view its property tax details.

This map was created by Sebastian Hallum Clarke, the author of Sidewalk Chorus. Countless experts and politicians have derided New York's unfair property tax system, but getting New York's legislators to act to fix the system first requires public awareness of the problems. This map is an attempt to help New Yorkers learn how their property tax system is failing them.

The code that powers this map is available on GitHub. All of the data that powers this map is from official City of New York sources. Read mode about the data.
